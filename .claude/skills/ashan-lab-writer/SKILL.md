# 阿珊-生成Lab详细内容

## Overview

根据课程大纲 + 项目源码，生成单个或全部 15 个 Lab 的完整 .md 文档。每个 Lab 是一份学生可以独立跟着操作的实验手册。

## 工作流程

```
课程大纲 (来自 ashan-course-generator)
    +
项目源码 (读取实际代码)
    ↓
生成 Lab .md 文档 → 保存到 docs/courses/{项目名}-Lab{N}-{标题}.md
```

## 每个 Lab 的固定模板

生成的每个 Lab .md 文件必须严格按以下结构：

```markdown

> **难度**：{⭐/⭐⭐/⭐⭐⭐}
> **前置 Lab**：Lab {N-1}（或"无"）
```

## 生成规则

### 1. 代码必须来自实际项目

读取项目源码，Lab 中的代码示例必须能跑。不要编造不存在的代码。

### 2. 验证步骤必须可执行

每个步骤写清楚"怎么验证你做对了"——比如浏览器打开什么地址、看到什么内容。

### 3. 坑点必须真实

从项目的 `CLAUDE.md` 中的"常见坑点"和实际代码中提取。不要凭空编造。

### 4. 保存规则

| 情况 | 保存路径 |
|------|---------|
| 生成单个 Lab | `docs/courses/{项目名}-Lab{N}-{标题}.md` |
| 批量生成所有 Lab | 逐个保存，全部放在 `docs/courses/` 下 |

### 5. 文件名规范

Lab 编号用两位数补齐：`Lab01`、`Lab02`...`Lab15`。

示例：`docs/courses/红薯快写-Lab01-项目介绍与环境搭建.md`

## 批量生成模式

当用户说"生成所有 Lab"时：

1. 先读取课程大纲文件（`docs/courses/{项目名}-课程大纲.md`）
2. 逐个生成 Lab 1-15
3. 每生成完一个 Lab，告知用户进度
4. 全部完成后，输出文件清单
