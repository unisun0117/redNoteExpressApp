# QDM-TOB 产品结构（按业务域划分）

## 角色与终端

| 角色 | 终端 | 定位 |
| :--- | :--- | :--- |
| **客户（下单人）** | 微信小程序 | 查商品、下单、支付、售后、充值、提现、领劵 |
| **业务员** | 企业微信移动端 | 发展客户、审核资料、拜访打卡，查看业绩完成情况 |
| **运营/管理员** | PC 后台 | 配置、管理、审批、数据统计 |

## 01-商品管理

统一管理商品数据，是交易管理的基础。

| 子模块 | 终端 | 说明 |
| :--- | :--- | :--- |
| 仓商品资料 | 后台 | 订购上限/下限/基数、自定义名称、商品库存 |
| 商品分类 | 后台 | 商品分类树管理 |
| 商品价格管理 | 后台 | 按仓库 + 商品维护销售价格 |
| 商品价格审核 | 企微 | 价格变更的审核详情与操作 |
| 商品搜索 | 小程序 | 搜索商品，加入购物车 |
| 商品分类页面 | 小程序 | 按分类浏览，加购 |

---

## 02-客户管理

从注册到建档的完整客户生命周期。

| 子模块 | 终端 | 说明 |
| :--- | :--- | :--- |
| 注册 | 小程序 | 哪些账号可以登录 |
| 登录 | 小程序 | 短信登录、微信授权登录 |
| 客户资料管理（公司）档案 | 小程序 | 登录账号绑定客户维护；可多个登录账户绑定同一个客户，资料提交后由原业务员审核 |
| 客户资料审核 | 企微 | 客户资料变更的审核详情与操作 |
| 登录账号管理 | 后台 | 小程序的注册用户管理，可绑定客户 |
| 客户资料管理（公司）档案 | 后台 | 收货地址管理，对接 SAP 创建客户，对接 TMS 收货要求 |

---

## 03-交易管理

从加购到售后的完整交易链路。

| 子模块 | 终端 | 说明 |
| :--- | :--- | :--- |
| 购物车管理 | 小程序 | 增减商品 |
| 购物车价格计算 | 小程序 | 计算优惠券、物流费用 |
| 支付结算界面 | 小程序 | 确认订单，可选到货日期 |
| 订单管理 | 小程序 | 我的订单列表及详情 |
| 退货管理 | 小程序 | 退款退货申请 |
| 销售订单管理 | 后台 | 订单管理、差补单、SAP 出库推送、SAP 分拣回执，接单后，推送SAP系统 |
| 退货单管理 | 后台 | 退货单管理，完成后需要推送SAP |

## 04-营销管理

为交易管理提供促销能力。

| 子模块 | 终端 | 说明 |
| :--- | :--- | :--- |

## 06-基础管理

统一账号体系、权限及系统配置，跨三个端复用。

| 子模块 | 终端 | 说明 |
| :--- | :--- | :--- |
| 用户管理 | 后台 | 管理可登录后台及企微移动端的用户账号，含角色绑定与数据权限配置 |
| 角色管理 | 后台 | 后台角色、企微移动端角色 |
| 数据权限管理 | 后台 | 后台/企微的数据级权限（如：业务员只看自己） |
| 字典管理 | 后台 | 系统/功能模块枚举值维护 |

## 07-运营管理

B2B 销售体系运营管理，含销售作业（客户拜访）。

| 子模块 | 终端 | 说明 |
| :--- | :--- | :--- |
| 消息管理 | 后台 | 小程序首页轮播公告管理 |
| 消息公告 | 小程序 | 公告列表及 HTML 内容展示 |
| 销售大区管理 | 后台 | 大区、开业时间、可下单天数、销售经理、基础规则 |
| 仓库管理 | 后台 | 仓库基本信息管理，添加、修改、删除 |
| 销售员管理 | 后台 | 销售员信息、每月业绩目标、邀请码 |
| 客户拜访记录 | 后台 | 业务员打卡数据查看 |
| 客户拜访打卡 | 企微 | 业务员上门打卡 |

| 子模块 | 终端 | 说明 |
| :--- | :--- | :--- |
| 待办管理 | 后台 | 业务员看自己的，非业务员按数据权限查看全部 |
| 待办列表 | 企微 | 业务员看自己的，非业务员按数据权限查看全部 |

## 09-数据管理

多维度经营数据归口统计。

| 子模块 | 终端 | 说明 |
| :--- | :--- | :--- |
| [商品统计报表](./09-数据管理/商品统计报表.md) | 后台 | 商品维度下单数量与金额，含多维筛选、列排序、单品下钻趋势图 |
| [客户统计报表](./09-数据管理/客户统计报表.md) | 后台 | 客户维度下单数量与金额统计，含6条件筛选、列排序、客户下钻趋势图 |
| [销售员统计报表](./09-数据管理/销售员统计报表.md) | 后台 | 销售员维度下单数量与金额统计，含两级筛选、8维度柱状图/折线图双模式、15/16列动态表格、团队均值对比、同比变化 |

## 10-公共服务管理

各模块共用的基础功能，不归属于具体业务域。

| 子模块 | 终端 | 说明 |
| :--- | :--- | :--- |
| 隐私政策管理 | 后台 | 富文本编辑维护 |
| 隐私政策展示 | 小程序 | 展示隐私政策 |
| 联系客服 | 小程序 | 对接七鱼 JS SDK |
| 首页 | 小程序 | 商品信息流（数据源归商品管理/营销管理） |
| 办公 | 企微 | 信息整合展示 |
| 我的 | 小程序 | 小程序账户个人信息整合 |

> **说明**：「办公」、「首页」和「我的」是页面容器，本身不承载独立业务逻辑，它们编排多个业务域的数据进行展示，因此归入公共服务管理。

## 小程序菜单目录

### 设计原则

> 客户使用路径：逛商品 → 加购 → 下单 → 管订单 → 售后。5 个底部 Tab，二级页面从 Tab 内进入。

### 底部 Tab

```
首页    分类    购物车    订单    我的
```

### 页面结构

```
🏠 首页                           📂 分类
├── 推荐商品 + 活动商品            ├── 分类列表（左侧）
├── 搜索入口                      ├── 商品列表（右侧）
├── 轮播公告                      └── → 商品详情
├── 快捷导航
└── 联系客服

🛒 购物车                         📋 订单
├── 商品列表（可增减）              ├── 全部订单（待支付/发货/收货/已完成）
├── 价格汇总                      └── → 订单详情
└── 去结算 → 支付结算页
    ├── 选地址
    ├── 选优惠券
    ├── 选到货日期
    └── 提交支付

👤 我的
├── 个人信息
├── 订单管理       ← 正常订单 + 售后订单
├── 收货地址
├── 我的优惠券
├── 我的钱包       ← 余额 / 充值 / 提现
└── 设置           ← 客户资料 / 联系客服 / 隐私政策 / 关于
```

```
买商品：首页/分类 → 商品详情 → 加入购物车 → 购物车 → 结算 → 支付
