# 新增功能：历史文章列表 + 点数动态同步

## 为什么要做

目前红薯快写 App 存在两个体验缺口：

1. **生成完文章没地方回看** — 虽然 Lab06-LAB08 已经修复了数据库存储（`generations` 表），但前端没有任何页面能查看历史生成记录。用户生成完一篇文章后，刷新页面就找不到了。

2. **点数显示不刷新** — 生成文章后右上角显示"剩余：9 次"，但这是页面加载时获取的旧数据。用户必须手动刷新页面才能看到最新点数。

## 做什么

### 功能 1：已生成文章列表
- 在"我的"页面（DashboardPage）新增"历史文章"tab/区域
- 调用后端 API 获取当前用户的所有生成记录
- 列表展示：标题 + 关键词 + 生成时间
- 点击可展开查看文章完整内容
- 支持一键复制

### 功能 2：点数动态同步
- 每次生成成功后，自动调用 `/api/auth/me` 刷新用户信息
- 右上角点数实时更新，不需要手动刷新页面

## 影响范围

- 后端：新增 `GET /api/generations` 接口
- 前端：修改 `DashboardPage.tsx`（新增历史列表）、修改 `GeneratorPage.tsx`（点数刷新）
- 课程：Lab07（前端生成页面）、Lab08（结果展示）、Lab10（点数计费）需要更新
