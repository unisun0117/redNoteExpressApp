📋产品概述
门店选址平台是一套面向零售门店拓展选址的全流程数字化管理系统，为拓展团队提供从城市沙盘分析、点位甄选、热力图评估、拓展任务指派、铺位考察到签约开店的完整业务闭环。
👔
拓展管理者
设定拓展目标、指派任务、审批评估、查看统计报表
🚶
一线拓展人员
录入拓展信息、执行任务、GPS 打卡、提交评估
🔍
选址分析人员
热力图分析、点位甄选、可达圈分析、品牌竞争分析

11权限管理 独立服务
权限同步服务（store-location-auth-syn），负责将本地权限数据推送到统一权限中心，同时通过阿里云 MQ 监听用户变更事件实现实时同步。

5 种权限类型

📋
PC 菜单权限
后台管理系统菜单控制
📱
移动端菜单权限
移动端菜单控制
🔘
按钮权限
页面操作按钮级控制
🏘️
数据权限
社区门店 / 铺位加盟商数据隔离

权限同步流程

本地 DB
→
凌晨4:00 定时任务
→
按序推送 HTTP API
→
统一权限中心
推送顺序：菜单 → 角色 → 数据维度 → 用户角色与数据权限

定时任务一览

执行时间任务方法功能说明
应用启动时init()补全用户信息 + 执行一次权限同步
凌晨 2:00completeUserInfo()从权限平台回填岗位、员工编号、邮箱等信息
凌晨 3:00refreshCache()刷新岗位 JetCache 缓存
凌晨 4:00synAuth()推送菜单/角色/数据/用户权限到统一权限中心
MQ 用户变更监听：两个独立线程分别监听 TOPIC_USER_INFO（新增/更新用户）和 REMOVEUSER_TOPIC（禁用/离职用户），消费后调用 SynPermissionService 更新本地权限。
