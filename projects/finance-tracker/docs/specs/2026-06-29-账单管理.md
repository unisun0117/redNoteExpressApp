# 账单管理

> 日期：2026-06-29
> 模块：transactions

## 功能

- 添加支出/收入记录
- 选择分类（支出8类 + 收入7类）
- 填写金额、日期、备注
- 账单列表（按日期分组、筛选、搜索）
- 编辑/删除账单

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/transactions | 列表（分页+筛选） |
| POST | /api/transactions | 新增 |
| PUT | /api/transactions/:id | 编辑 |
| DELETE | /api/transactions/:id | 删除 |

## 前端页面

- 记账表单页（支出/收入切换，分类网格）
- 账单列表页（按日期分组，筛选标签，搜索框）
