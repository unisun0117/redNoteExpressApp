# 用户认证

> 日期：2026-06-29
> 模块：auth

## 功能

- 邮箱 + 密码注册
- 邮箱 + 密码登录
- JWT Access Token + Refresh Token
- Token 自动刷新
- 路由守卫（未登录跳登录页）

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/auth/register | 注册 |
| POST | /api/auth/login | 登录 |
| POST | /api/auth/refresh | 刷新token |
| GET | /api/auth/me | 获取当前用户信息 |

## 前端页面

- 登录/注册页（tab切换）
- 密码可见性切换
- 登录成功跳转首页
