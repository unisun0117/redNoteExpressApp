# 个人中心

> 日期：2026-06-29
> 模块：profile

## 功能

- 个人信息展示（头像、昵称、邮箱、加入时间）
- 累计数据（总笔数、累计收入、累计支出）
- 数据导出
- 设置菜单
- 退出登录

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/user/profile | 个人信息 |
| PUT | /api/user/profile | 编辑资料 |
| GET | /api/user/stats | 累计统计 |

## 前端页面

- 个人中心页（头像卡片 + 数据概览 + 菜单列表 + 退出按钮）
