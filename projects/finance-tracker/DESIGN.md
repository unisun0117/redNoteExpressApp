# DESIGN.md — 金橘记账

## 品牌调性
- **一句话定位：** 温暖治愈的个人记账工具，让理财不再冷冰冰
- **情绪关键词：** 温暖、安心、清晰、轻量、治愈
- **参考设计：** Linear（简洁）+ Notion（温暖）+ 支付宝（可信赖）

## 色彩系统

| Token | 色值 | 用途 |
|--------|------|------|
| `--color-primary` | `#F59E0B` (琥珀) | 主按钮、选中态、收入标识 |
| `--color-primary-light` | `#FEF3C7` | 主色浅底 |
| `--color-expense` | `#EF4444` (暖红) | 支出标识、删除按钮 |
| `--color-income` | `#10B981` (翠绿) | 收入标识 |
| `--color-bg` | `#FFF8F0` | 页面背景（暖白） |
| `--color-card` | `#FFFFFF` | 卡片背景 |
| `--color-text` | `#1F2937` | 主文字 |
| `--color-text-secondary` | `#9CA3AF` | 次要文字 |
| `--color-border` | `#F3E8DC` | 分割线/边框 |

## 字体

| 层级 | 字号 | 字重 | 用途 |
|------|------|------|------|
| H1 | 24px | 700 | 页面标题 |
| H2 | 18px | 600 | 区块标题 |
| Body | 15px | 400 | 正文 |
| Caption | 12px | 400 | 辅助说明 |
| Number | 28px | 700 | 金额数字（DIN Alternate 风格） |

## 圆角 & 阴影

- 卡片圆角：12px
- 按钮圆角：8px
- 输入框圆角：8px
- 卡片阴影：`0 1px 3px rgba(0,0,0,0.06)`

## 间距系统（4px 基准）

- xs: 4px
- sm: 8px
- md: 12px
- lg: 16px
- xl: 24px
- 2xl: 32px

## 动效

- 按钮 hover：`transform: translateY(-1px)` + 阴影加深
- 列表项 hover：背景色过渡 150ms
- 金额变化：数字跳动动画（可选）
- 页面切换：无动画（极简）

## 组件规范

### 金额显示
- 支出：`- ¥1,200.00`（红色）
- 收入：`+ ¥3,500.00`（绿色）
- 余额：大字显示，无符号，深色

### 分类图标
- 使用 emoji 代替图标库（减少依赖）
- 餐饮 🍽️ 交通 🚇 购物 🛍️ 娱乐 🎬 住房 🏠 医疗 💊 教育 📚 其他 📌

### 空状态
- 灰色插画 + "还没有记录" + 引导按钮
