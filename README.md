# AI_Workspace
AI 自动化、机器学习、OpenSpec 项目工作台

## 分支说明
### 1. main/master（全局主干）
- 存放全局共享技能：superpower/
- 所有项目均可直接使用
- 仅存放公共内容，不编写业务代码

### 2. docs/csdn_articles
CSDN 博客文章、稿件、素材归档

### 3. data_analysis/machine_learning
数据分析与机器学习、算法实验、数据处理

### 4. agent/hermes_agent
Hermes Agent 智能体开发、技能库、框架配置

### 5. openspec/* （OpenSpec 独立项目）
每个项目独立安装 OpenSpec，共享 main/master 分支的全局 superpower
- openspec/fullstack-blog
- openspec/kitchen-timer
- openspec/redNoteExpress
