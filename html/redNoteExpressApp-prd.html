1️⃣ 产品概述

一句话定位

上传图片 + 输入关键词 + 选择风格 → AI 生成一篇完整小红书风格文章。

解决的痛点

痛点现状红薯快写方案
不会写小红书文案对着空白编辑器发呆上传图片，AI 自动生成结构化文章
风格拿捏不准写出来像说明书不像种草笔记内置 20+ 赛道风格模板
文章结构松散想到哪写到哪固定 7 段式：标题→前言→3段落→总结→店铺
文案效率低一篇要写 30 分钟30 秒生成，微调即发

6️⃣ API 设计

核心接口

方法路径鉴权说明
POST/api/auth/register无邮箱 + 密码注册
POST/api/auth/login无返回 access_token + refresh_token
POST/api/auth/refreshRefresh Token刷新 access_token
GET/api/auth/meBearer当前用户信息 + 剩余点数
POST/api/generateBearer核心接口：上传图片 + 参数 → 生成文章
GET/api/styles无返回模板列表和赛道列表
POST/api/viral/analyzeBearer + VIP爆款分析（桩实现）
GET/api/billing/packages无点数包列表（桩实现）
POST/api/batch/generateBearer + VIP批量生成（桩实现）

POST /api/generate 请求参数

参数类型必填说明
imageFile (multipart)是产品图片（单张）
keywordsString是产品名/卖点/场景关键词
trackString否内容赛道，默认 "general"
styleString否风格模板，默认 "种草"
use_emojiBoolean否是否使用 Emoji，默认 true
use_subtitlesBoolean否是否使用小标题，默认 true

错误码

HTTP 状态码含义
200成功返回文章 JSON
401Token 过期或无效
402点数不足，需购买
403VIP 专属功能，当前用户非 VIP
422参数校验失败
500AI 生成失败（DeepSeek API 错误）

🔟 非功能需求

性能

指标目标
页面首次加载< 3s (Vercel CDN)
API 响应时间（生成）< 30s（取决于 DeepSeek）
API 响应时间（其他）< 500ms
并发用户50+ （PythonAnywhere 免费版限制）

安全

  • JWT access_token 15 分钟过期 + refresh_token 7 天
  • 密码 bcrypt 哈希存储（cost factor=12）
  • 图片上传大小限制 5MB
  • CORS 白名单控制
  • SQLAlchemy 参数化查询防注入

可用性

  • 前端 Vercel 自动部署（git push → 上线）
  • 后端 PythonAnywhere 自动拉取（Webhook）
  • DeepSeek API 故障时返回友好错误提示
  • 点数不足时引导用户购买而非直接报错

兼容性

  • 支持 Chrome / Safari / Edge 最新两个版本
  • 移动端响应式布局（antd-mobile）
  • 微信内置浏览器兼容
