🔍 什么是 Agent-Reach？

💡

Agent-Reach 是一个基于 MCP（Model Context Protocol） 的 AI 智能体联网触达层。 它让本地 AI Agent 突破信息孤岛，无需任何平台 API Key 即可直接读取和搜索 Twitter、Reddit、YouTube、GitHub、Bilibili、小红书等主流平台的内容。

🏝️

打破信息孤岛

本地 AI Agent 默认无法访问互联网。Agent-Reach 作为 MCP 服务桥接层，让 AI 实时获取全网公开信息。

🔑

零 API 费用

不需要申请任何平台的开发者 Key，不需要付费 API 套餐。直接通过 CLI 抓取公开页面内容。

🔌

一行命令接入

通过 MCP 协议标准接口，Claude Code、Cursor、Cline 等 AI 工具一键集成。

⚡ 核心能力

🔎

平台搜索

在 Reddit、Twitter、小红书等平台执行关键词搜索，返回结构化结果。

📄

内容提取

抓取网页正文、社交媒体帖子、GitHub README、YouTube 字幕等，自动清洗为纯文本/Markdown。

🧵

上下文理解

获取 Reddit 帖子下的评论树、Twitter 线程、GitHub Issue 讨论，支持嵌套上下文。

🔧

CLI 原生

基于命令行工具链，轻量高效，对系统资源占用极低，支持无头浏览器模式。

🧩

Skill 系统

内置可扩展的 Skill 机制，可按需加载平台适配器，灵活组合使用场景。

🛡️

隐私友好

本地运行，数据不过第三方服务器。仅访问公开内容，遵守 robots.txt 和平台规范。

🚀 快速开始

安装 Agent-Reach

通过 pip 或 npm 全局安装 MCP Server 包

# Python pip install agent-reach-mcp # 或 Node.js npm install -g agent-reach-mcp

配置 MCP 客户端

在 Claude Code / Cursor / Cline 的 MCP 配置中添加 Agent-Reach

{ "mcpServers": { "agent-reach": { "command": "npx", "args": ["-y", "agent-reach-mcp"] } } }

直接使用

AI Agent 现在可以直接搜索和读取互联网内容

# 用户对 AI 说： "帮我在 Reddit 上找关于 GPT-5 的最新讨论" "搜索小红书上的露营攻略并总结" "看看这个 GitHub 项目的 README"
